AngelBot

A Discord bot that likes to communicate with APIs and started with FF XIV and XIVDB.
Add Bot


settings

Per Server Settings

While the bot started on one server, a lot of feedback came from that server. Not unlike the dynamic rules that other bots implement, this bot has per server rules. You can even lock individual commands to individual channels!

help

Well Documented

Do you need help? There are help files available at RTFD. Always up to date with the latest code to inform you of any pressing changes and always maintained by the developer of AngelBot.

apps

Customizable

Want to add your own module? Yeah I can dig that. The modules have a standard format that is pretty apparent and a documentation page that explains further details. Then just submit a request on github at AngelBot and we can get your module added.

network_wifi

APIs

The focus of the bot is entirely on external APIs. While other bots focus on information available within Discord or parsing websites, AngelBot parses APIs. Currently there is support for XIVDB with coding already started on Github and Anilist and plans for LCBO, Hummingbird.me and Twitch.tv.

child_friendly

Bot Friendly

With the ability to customize the command prefix that the bot responds to per server and the fact that the admin commands use prefixes that should never ever be copied, this bot is extremely friendly with other bots and shouldn't ever interfere or dual fire.

account_circle

OAuth

That's right, this bot can get to know you. It can log into your accounts. It supports OAuth 2 by redirecting you to your familiar website flow and then collecting the results here at angelbot HQ.



About AngelBot

AngelBot was made using Discord.py. It is open source and the code is available on Github. You can contact AngelBot's developer on Discord by finding Rory in Discord-dev or AngelBot. This bot is also listed on Carbon.
